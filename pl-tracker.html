🏷 Categories & Tax Treatment

This controls how each category is treated in the P&L Report — exactly how QuickBooks uses account types so that transfers, owner draws, loan principal, card payments and taxes don’t get wrongly deducted from profit. Settings here are per-client — each customer can treat the same category differently, and any category you leave alone follows the shared default. For the partial options (partially deductible / partially taxable) you can type any percent from 0–100; the rest is automatically tracked off the P&L. Changes recalculate this client’s report instantly. Nothing else (the Ledger, raw records) is altered.
